# Data Archive

Historical data storage for PM Radar.

## Purpose

This directory stores historical research data for:
- Week-over-week trend analysis
- Historical context for investigations
- Long-term pattern detection

## Structure

Raw JSON files and reports are stored in parent directories:
- `/data/raw/` - Weekly collection JSON files
- `/data/reports/` - Weekly markdown reports
- `/data/competitor-maps/` - Competitor site structure maps

Files older than 3 months can be archived here if needed.
