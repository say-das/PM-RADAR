# Data Directory

This directory contains runtime data generated by PM Radar.

## Structure

```
data/
├── raw/              # Collected JSON data from RSS and Reddit
│   ├── YYYY-MM-DD.json           # Daily collection (RSS + Reddit combined)
│   └── YYYY-MM-DD-analysis.json  # GPT-4o analysis results
│
├── reports/          # Generated reports
│   ├── YYYY-MM-DD.md             # Markdown report
│   └── YYYY-MM-DD.html           # HTML report (for email)
│
├── archive/          # Old reports (manual archiving)
│
└── competitor-maps/  # Competitor URL mappings (future feature)
```

## Gitignore Status

Most files in this directory are **gitignored** to avoid committing generated data and reports:

✅ **Tracked in git:**
- `README.md` (this file)
- `.gitkeep` files (to preserve folder structure)

❌ **Ignored in git:**
- `raw/*.json` - Collected data
- `reports/*.md` - Markdown reports
- `reports/*.html` - HTML reports
- `reports/*.html.bak` - Backup files
- `competitor-maps/` - Competitor data

## Data Retention

**Local development:**
- Keep last 7 days of data for debugging
- Archive older reports manually if needed

**GitHub Actions:**
- Data is regenerated each run (not persisted)
- Reports are emailed, not stored

## Manual Cleanup

To clean up old data:

```bash
# Remove data older than 7 days
find data/raw -name "*.json" -mtime +7 -delete
find data/reports -name "*.md" -mtime +7 -delete
find data/reports -name "*.html" -mtime +7 -delete
```

## Example Files

**Collection file** (`data/raw/2026-04-09.json`):
```json
{
  "collected_at": "2026-04-09T17:00:00",
  "rss_articles": [...],
  "reddit_posts": [...]
}
```

**Analysis file** (`data/raw/2026-04-09-analysis.json`):
```json
{
  "analyzed_at": "2026-04-09T17:01:00",
  "telecom_fraud_summary": "...",
  "general_fraud_summary": "...",
  "reddit_community_summary": "..."
}
```

**Report file** (`data/reports/2026-04-09.md`):
- Markdown-formatted report with sections
- Inline citations to sources
- Glossary and bibliography

**HTML file** (`data/reports/2026-04-09.html`):
- Styled HTML for email delivery
- Can be printed to PDF via browser
